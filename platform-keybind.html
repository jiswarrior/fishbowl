Playing Together Across Platforms

Palia supports full cross‑play and cross‑progression between PC and Nintendo Switch. This guide lists default keybinds for both platforms and explains how to add friends, join parties, and sync your progress seamlessly.

🌐 Cross-Play & Cross-Progression

  • Cross‑play is automatic: PC and Switch players share the same servers. You can party up with anyone regardless of platform.
  • Cross‑progression: Your account progress (skills, inventory, home, quests) is saved on Palia’s servers. Log into the same account on PC or Switch, and you’ll have everything.
  • Linking accounts: If you start on PC, you can later link your Nintendo account via the Palia website to continue on Switch. Progress is shared.
  • Friend list sync: Friends added on PC also appear on Switch (and vice versa). You must have an active Palia account on both platforms with same email.
💡 Voice chat is not built‑in; use Discord (PC or mobile) for cross‑platform voice.

👥 How to Add Cross‑Platform Friends & Invite to Party

  1. Send a friend request: Press O (PC) or hold ZL + A (Switch) → “Add Friend”. Type the player’s exact in‑game name (case‑sensitive).
  2. Accept request: The recipient must accept via their social panel.
  3. Invite to party: Once friends, open party menu → “Invite to Party”. Cross‑platform works seamlessly.
  4. Join via code (alternative): If direct invite fails, use the “Copy Invite Code” feature in the party menu and share it via text/chat. Other players can join by entering that code.

Troubleshooting: If friend doesn’t appear, ensure both players are online on the same server region (NA/EU/Asia).

🔧 Switch‑Specific Recommendations

  • Text chat: Using the on‑screen keyboard is slow. Consider using the Nintendo Switch Online mobile app for voice chat instead of typing.
  • Performance mode: In TV mode, set Switch to 720p output for smoother frame rates.
  • Pro Controller recommended – Joy‑Cons are usable but less comfortable for long sessions.
  • Battery life: Palia drains battery faster than average; keep the charger nearby.

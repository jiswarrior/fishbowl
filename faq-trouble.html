Solving Technical Hiccups

Even a cozy game like Palia can run into performance or connectivity issues. This guide helps you diagnose and fix common problems – from lag and ping spikes to ensuring your progress is never lost.

🐢 Game is laggy or low FPS

For PC players:

  • Lower graphics settings: Shadows → Low, Anti‑aliasing → Off, View Distance → Medium. (See the Performance Tuning guide for detailed presets.)
  • Close background applications (browsers, Discord, Spotify).
  • Update graphics drivers (NVIDIA/AMD/Intel).
  • Switch to Fullscreen mode instead of Windowed.
  • Lower resolution to 720p or 900p if needed.
  • Disable V‑Sync and set FPS cap to 60 or 30.

For Nintendo Switch:

  • Restart the console (hold power → Restart).
  • Play in Docked mode with a wired LAN adapter for stability.
  • Lower TV resolution to 720p (in Switch System Settings).
  • Close other software (press Home, close any open games).
  • Install the game on internal storage (not SD card) for slightly faster loading.
💡 If lag happens only in certain zones (e.g., Bahari Bay with many players), try switching servers by teleporting home and back, or relogging.

🌐 Frequent disconnects or high ping

  • Use a wired connection – Ethernet is always more stable than Wi‑Fi.
  • Check your region – In the launcher, select the server region closest to you (NA, EU, Asia). Playing on a distant region increases ping.
  • Restart your router/modem – A classic but effective fix for most network issues.
  • Disable VPN or proxy – These can interfere with game traffic.
  • Port forwarding (advanced) – Forward UDP ports 27000–27030 to your PC. This may improve connectivity on strict NAT networks.
  • DNS change – Use Google DNS (8.8.8.8 / 8.8.4.4) or Cloudflare (1.1.1.1) for potentially faster resolution.
  • Check server status – Visit the official Palia Discord or server status page for maintenance announcements.

Switch specific: Go to System Settings → Internet → Test Connection. If NAT type is D or worse, try connecting via a mobile hotspot to bypass strict NAT.

💾 How to back up my save? (PC)

Palia uses cloud saves (server‑side). Your progress is tied to your account – you do not need to manually back up, and you can play from any device after logging in.

However, for extra safety or to transfer settings/configurations, you can copy local configuration files (not the save itself):

  • Location: %LOCALAPPDATA%\Palia\Saved\Config\WindowsClient\
  • Files: GameUserSettings.ini (graphics/controls) and Input.ini (keybinds).
  • Backup these files manually if you plan to reinstall Windows or move to another PC.

Switch users: Save data is automatically backed up to Nintendo’s cloud if you have a Nintendo Switch Online subscription. Otherwise, the save remains on the console.

⚠️ There is no “local save file” for your inventory or skill levels – everything is on Palia’s servers. Deleting local files will NOT delete your progress.

🚨 Still having issues?

  • Verify game files (PC): In the Palia launcher, go to Settings → Verify Integrity.
  • Reinstall the game – Corrupted files can cause strange bugs.
  • Contact official support: Visit support.palia.com or use the in‑game “Report Bug” (Esc → Support).
  • Community help: Ask in #help channel on the official Palia Discord – players and moderators are responsive.

🔍 Quick Diagnostic Steps (PC)

  1. Check your internet speed and ping (speedtest.net).
  2. Update graphics drivers and Windows.
  3. Lower in‑game graphics settings.
  4. Close other programs (especially Chrome/Discord).
  5. Restart the game and launcher.
  6. Restart your PC and router.
  7. Reinstall the game as last resort.
